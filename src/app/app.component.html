Customizing list items

Intro

The list component is from a library that we control. It renders data from an input array into items. A new requirement comes along where we need to wrap some of the list items in component that we get from another library, which we don't control. Additionally, we use our library in other projects that don't need to wrap the list items.

This means that our library needs to give client applications control over the wrapping code.

Demo section

This demo section shows how to render data from the input array in a list component that allows the embedding code to provide its own wrapper.
